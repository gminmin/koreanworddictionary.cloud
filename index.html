사용 방법

1. 설정에서 Gemini API 키와 표준국어대사전 API 키를 입력하세요.

2. 찾고 싶은 단어의 의미나 상황을 설명해주세요.

3. Gemini가 적절한 한국어 단어를 찾아드립니다.

정보

개발: Gemini 2.0 Flash + 표준국어대사전 API

페키지에 포함된 api_test.html를 이용하여 지원이 가능한지 확인할 수 있습니다.

버전: 1.0.0

문제 해결

• 단어를 찾지 못하는 경우: API 키가 올바른지 확인하세요.

• 일부 단어가 표시되지 않는 경우: 프록시 서버 문제일 수 있습니다. 다시 시도해보세요.
